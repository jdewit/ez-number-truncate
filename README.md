ez-number-truncate
==================

Angular number truncate filter

###Example

```
{{ 3000 | ezNumberTruncate }} // -> 3000
{{ 300000 | ezNumberTruncate }} // -> 300K
{{ 300000000 | ezNumberTruncate }} // -> 300Mil
{{ 3550000 | ezNumberTruncate }} // -> 35.5Mil
{{ 3500000 | ezNumberTruncate }} // -> 3.5Mil
```
